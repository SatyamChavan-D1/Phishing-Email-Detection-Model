# Phishing Email Detection Model

This project uses Machine Learning to detect phishing emails.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- TF-IDF
- Logistic Regression

## Features
- Detects phishing emails
- Trains on email dataset
- Predicts custom emails

## Run Project

Install packages:

pip install -r requirements.txt

Run:

python phishing_detector.py
